# Weather App

A simple weather app that displays the weather of a city.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- OpenWeatherMap API

## How to Run
Clone the repo by using the following command:

```bash
git clone <repo_url>
```

After cloning the repo, run the following command to install the dependencies:

```bash
npm install
```

Finally, run the following command to start the development server:

```bash
npm run dev
```
